# 🚀 Task Tracker (MERN Stack Project)

A full-stack Project & Issue Tracking system built using the MERN stack.
Users can manage projects, create issues, assign tasks, and track progress.

---

## 🔥 Features

* 🔐 User Authentication (Login/Register with JWT)
* 📁 Create & Manage Projects
* 🐞 Create, Assign & Track Issues
* 📊 Dashboard with Project & Issue Stats
* 🔎 Filters (Status, Priority, Project)
* 👥 Role-based functionality (Admin / Member)
* ⚡ Responsive UI

---

## 🛠 Tech Stack

* **Frontend:** React.js, Axios, CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas
* **Auth:** JWT (JSON Web Token)

## ⚙️ Setup Instructions (Local)

### 1️⃣ Clone the repo

```bash
git clone https://github.com/ssrohan24/Task-Tracker.git
cd Task-Tracker
```

---

### 2️⃣ Backend Setup

```bash
cd backend
npm install
npm run dev
```

Create `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
```

---

### 3️⃣ Frontend Setup

```bash
cd frontend
npm install
npm start
```
